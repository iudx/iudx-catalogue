Catalogue Details

  • A catalogue item is a JSON object
  • Each item is described by a schema which is specified using JSON-schema
    • Used to perform type-checking for every item inserted in catalogue
    • Schemas are extendible
    • Schemas are also stored in catalogue
      • Searchable
  • Every item schema extends a schema from a set of “base-schemas”
    • Way to mandate a few required information fields for each item
    • A catalogue item provider can extend a base schema to provide additional fields
